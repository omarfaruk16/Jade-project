# Jade Project

A premium Full-stack template.

## Tech Stack
- **Frontend**: Next.js 15 (TypeScript, App Router)
- **Backend**: Express.js
- **Styling**: Vanilla CSS (Premium Design)

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation

1. Install dependencies for root:
   ```bash
   npm install
   ```

2. Install dependencies for Frontend:
   ```bash
   cd frontend && npm install
   ```

3. Install dependencies for Backend:
   ```bash
   cd backend && npm install
   ```

### Running the Project

Run both frontend and backend concurrently:
```bash
npm run dev
```

The frontend will be available at [http://localhost:3000](http://localhost:3000) and the backend at [http://localhost:5000](http://localhost:5000).
